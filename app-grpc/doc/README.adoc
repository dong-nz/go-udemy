== Generate gRPC code
Go to app-grpc directory
[source]
----
protoc --proto_path=./proto --go_out=./src/gen --go_opt=paths=source_relative --go-grpc_out=./src/gen --go-grpc_opt=paths=source_relative proto/*.proto
----
